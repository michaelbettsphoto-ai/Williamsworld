# Background Music

This directory contains background music loops for Williams World.

## Sound Files

- `hub-loop.ogg` - Light town hub music (low-intensity)
- `forest-loop.ogg` - Light forest path music (low-intensity)
- `dungeon-loop.ogg` - Light dungeon corridor music (low-intensity)

## Technical Specs
- Format: OGG (loopable)
- Volume: Low-intensity, quiet (default 0.2)
- Duration: Seamless loops
- Style: Adventurous, never scary, fits "Pok√©mon x D&D" vibe

## Behavior
- Music is optional and can be controlled separately from SFX
- Users can adjust music volume independently
- Music never overpowers UI sound effects
- Should loop seamlessly without noticeable gaps
